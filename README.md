# CodeAlpha_Stock-Portfolio-Tracer

## 📌 Project Overview
This project was developed as part of my internship at **CodeAlpha**.  
The goal is to build a simple **Stock Portfolio Tracker** that calculates the total investment value based on user‑defined stock quantities and hardcoded stock prices.

---

## 🎯 Features
- Interactive input: user enters stock symbols and quantities.
- Hardcoded dictionary for stock prices.
- Calculates total investment value.
- Displays portfolio summary.
- Saves results to a `.txt` file (UTF‑8 encoded).

---

## 🛠️ Key Concepts Used
- **[Dictionary](ca://s?q=Python_dictionary_usage)** → to store stock prices and portfolio data.  
- **[Input/Output](ca://s?q=Python_input_output_in_Python)** → user input for stocks and quantities, file output for saving results.  
- **[Basic Arithmetic](ca://s?q=Python_basic_arithmetic_operations)** → multiplication and summation for investment calculation.  
- **[File Handling](ca://s?q=Python_file_handling_write_mode)** → writing results to a text file.
